Код -це текст за правила спец.синтаксису.
Розробка має певний цикл: планування, аналіз, дизайн, розробка,тестування, підтримка.
Scrum process - підход до розробки, правила для ефективної праці в команді.
Fronted розробка - три технології які працюють в браузері: HTML(розмітка), CSS(відображення), JS(взаємодія)
VCS Version Control Sistem (Системи контролю версій)
Git - це система яка дозволяє працювати в команді, мати історію змін файлів і кому ці зміни належать, зберігати попередні версії
